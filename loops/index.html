Exercicio 1

O que o código abaixo está fazendo? Qual o resultado impresso no console?
let valor = 0
for(let i = 0; i < 5; i++) {
valor += i
}
console.log(valor)

Respostas

1- o código está fazendo um loop começando com o número "0" e terminando no "4"
2- a cada loop pega o número seguinte
3- ele vá somar os valores de 0 + 1 + 2 + 3 + 4 = 10 e guardar na let "valor"
4- console.log imprimirá o valor de 10
